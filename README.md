# SAF
Secret Account Finder (SAF) for Twitter, writing in Python 3.x.
This finds users who have similar FF networks.

***

### Input:
User list will be followed by target user
  - @hoge
  - @fuga
  - @piyo
  - ...

### Output:
Users following input users

**85.24%**
- @user1
- @user2
- @user3

**53.76%**
- @user4
- @user5

**23.66%**
- @user6
